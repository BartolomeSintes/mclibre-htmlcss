Esta página muestra las 2231 entidades de carácter HTML incluidas en la norma HTML del WHATWG (y que ya estaban incluidas en la recomendación HTML 5.2 de diciembre de 2017).

Antes de leer esta lección se recomienda leer las lecciones:

  • El juego de caracteres UTF-8, que explica qué es un juego de caracteres informático y, concretamente, qué es el juego de caracteres UTF-8

Después de esta lección se recomienda leer la página Dibujos en Unicode, que muestra los caracteres Unicode que corresponden a dibujos (emoticonos, emojis, etc.)

Información contenida en las fichas

Para cada entidad de carácter se muestra:

  • el carácter representado por la entidad de carácter
  • la entidad de carácter (con letras)
  • la entidad de carácter (código decimal)
  • la entidad de carácter (código hexadecimal)
  • la versión de HTML en la que se incluyó la entidad
  • descripción del carácter (en su caso)
