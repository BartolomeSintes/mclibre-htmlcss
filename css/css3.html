Recomendaciones CSS 3

Así como CSS 2.1 es una única recomendación, para la elaboración de CSS 3 el W3C eligió un enfoque modular: CSS 3 está dividida en numerosas recomendaciones que abordan cada una un aspecto distinto de las hojas de estilo.

Cada módulo CSS 3 sigue su propio ritmo de desarrollo y los navegadores también siguen su propio ritmo de implementación de las recomendaciones o de sus borradores. En la web del W3C se pueden consultar:

  • un listado detallado de las recomendaciones CSS vigentes y en elaboración
  • listado breve de las recomendaciones CSS vigentes y en elaboración
  • listado detallado de los borradores
  • CSS Indexes, un índice de todos los términos definidos por las recomendaciones CSS

Muchas propiedades que aparecen en los borradores de las futuras recomendaciones de las hojas de estilo CSS 3 ya funcionan en las versiones actuales de los navegadores, pero no lo hacían en las versiones antiguas (especialmente en Internet Explorer 11, el único navegador antiguo que todavía mantiene actualmente -noviembre de 2019- una cuota de mercado significativa, aunque sea residual). Por ello, al diseñar un sitio web, el diseñador debe tener en cuenta los navegadores utilizados por los usuarios de ese sitio y decidir en qué momento puede dejar de preocuparse por una determinada versión de un navegador.

Recomendaciones CSS 3 aprobadas

Actualmente (noviembre de 2019), el W3C ha aprobado las siguientes recomendaciones CSS 3:

  • junio de 2011: CSS 3 Color, editada por Tantek Çelik y otros. Reemplazada en junio de 2018 por la 2º edición.
  • septiembre de 2011: CSS Espacios de nombres (CSS Namespaces), editada por Elika J. Etemad. Modificada en marzo de 2014 - versión de 2011.
  • septiembre de 2011: CSS 3 Selectores, editada por Tantek Çelik y otros. Reemplazada en noviembre de 2018 por la 2º edición.
  • junio de 2012: Media Queries, editada por Håkon Wium Lie y otros.
  • noviembre de 2013: CSS 3 Atributos style (CSS Style Attributes), editada por Tantek Çelik y Elika J. Etemad.
  • junio de 2018: CSS 3 Color (2º ed), editada por Tantek Çelik.
  • junio de 2018: CSS 3 Basic User Interface, editada por Tantek Çelik y Florian Rivoal.
  • septiembre de 2018: CSS Fonts Module Level 3, editada por John Daggett y otros.
  • noviembre de 2018: Selectors Level 3 (2º ed), editada por Tantek Çelik y otros.

Notas de grupo

Algunos documentos se han publicado como notas de grupo de trabajo (Working group notes) en vez de como recomendaciones. Eso quiere decir que el grupo de trabajo correspondiente considera que no vale la pena seguir trabajando en ese tema y publica su último borrador como nota de grupo de trabajo.

  • marzo de 2013: CSS: Perfil Impresora (CSS Print Profile)
  • octubre de 2014: CSS: Perfil TV 1.0 (CSS TV Profile 1.0)
  • octubre de 2014: CSS: Perfil móvil (CSS Mobile Profile 2.0)
  • octubre de 2014: CSS3: Marquee (CSS Marquee Module Level 3)
  • octubre de 2014: CSS3 Hyperlink Presentation Module
  • octubre de 2014: The CSS Reader Media Type
  • octubre de 2014: CSS Presentation Levels Module
  • octubre de 2014: Behavioral Extensions to CSS
  • noviembre de 2014: Fullscreen
  • marzo de 2015: CSS Template Layout Module
  • junio de 2018: CSS Speech Module
  • abril de 2019: Non-element Selectors Module Level 1

Recomendaciones CSS 3 en elaboración

Actualmente (noviembre de 2019), el W3C está elaborando un gran número de recomendaciones CSS 3. Por su parte, los navegadores están implementando muchas de estas recomendaciones aunque todavía estén en elaboración.
