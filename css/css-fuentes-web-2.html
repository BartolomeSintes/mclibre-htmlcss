En esta lección se comentan algunas de las propiedades CSS definidas en la futura recomendación CSS Fonts Module Level 4. Actualmente (noviembre de 2021) esta recomendación se encuentra en elaboración y no se considera parte de la definición oficial de CSS en CSS Snapshot 2021, aunque se espera que puedan hacerlo en el futuro.

En esta lección se comentan las propiedades relacionadas con las fuentes variables. Aunque dentro de unos años seguramente se podrán utilizar estas propiedades con las familias de fuentes genéricas instaladas en el ordenador (serif, sans-serif, etc.), hasta entonces la mayoría de fuentes variables se obtendrán mediante el mecanismo de las fuentes web, que se comenta en la lección CSS Fuentes web.
