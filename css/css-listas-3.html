En la lección CSS Listas (1) se comentan propiedades CSS definidas en la recomendación CSS 2.1, publicada en junio de 2011. En esta lección se comentan propiedades CSS definidas en la recomendación CSS 2.1, publicada en junio de 2011, y en la futura recomendación CSS Counter Styles Level 3. Aunque actualmente (mayo de 2022) esta recomendación no está formalmente aprobada, se considera parte de la definición oficial de CSS en CSS Snapshot 2021.

Estas propiedades se aplican a las listas (<ul>, <ol>), pero también a cualquier elemento que tenga la propiedad display con el valor list-item, como por ejemplo el elemento <summary>.

Tipo predefinidos de marcador: list-style-type

list-style-type

El marcador de las listas ordenadas (<ol>) o no ordenadas (<ul>) se modifica con la propiedad list-style-type.

En la lección CSS Listas (1) se muestran los tipos de marcador incluidos en la recomendación CSS 2.1. En esta lección se muestran los tipos de marcador incorporados en la futura recomendación CSS Counter Styles Level 3, hebrew, cjk-ideographic, hiragana, katakana, hiragana-iroha, katakana-iroha, correspondientes a lenguas orientales.

No se muestran los siguientes tipos de marcador: arabic-indic, upper-armenian, lower-armenian, bengali, cambodian, khmer, cjk-decimal, devanagari, gujarati, gurmukhi, hebrew, kannada, lao, malayalam, mongolian, myanmar, oriya, persian, lower-roman, upper-roman, tamil, telugu, thai, tibetan, disclosure-open, disclosure-closed, cjk-earthly-branch, cjk-heavenly-stem.

ul {
  list-style-type: hebrew;
}

ul {
  list-style-type: georgian;
}

ul {
  list-style-type: armenian;
}

ul {
  list-style-type: cjk-ideographic;
}

ul {
  list-style-type: hiragana;
}

ul {
  list-style-type: katakana;
}

ul {
  list-style-type: hiragana-iroha;
}

ul {
  list-style-type: katakana-iroha;
}
