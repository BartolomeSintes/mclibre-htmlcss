La definición oficial de las hojas de estilo CSS se encuentra actualmente (diciembre de 2021) en el documento CSS Snapshot 2021. Esta lección enlaza las últimas versiones de las recomendaciones o de sus borradores que forman parte de esa definición. En la lección Recomendaciones CSS inestables se enlazan las últimas versiones de los borradores que todavía no forman parte de la definición oficial de CSS.

Definición oficial actual de CSS

Así como CSS 2.1 es una única recomendación, para la elaboración de CSS 3 el W3C eligió un enfoque modular: CSS 3 está dividida en numerosas recomendaciones que abordan cada una un aspecto distinto de las hojas de estilo. Cada módulo CSS 3 sigue por tanto su propio ritmo de desarrollo, al mismo tiempo que los navegadores también siguen su propio ritmo de implementación de las recomendaciones o de sus borradores. Si a esto añadimos que de algunas recomendaciones se han publicado borradores de versiones CSS 4 o CSS 5, y que el W3C en 2020 incoporó un modelo de recomendación continua en el que ya no es necesario publicar formalmente la recomendación, la situación puede ser bastante confusa.

Para clarificar un poco esta situación, el grupo de trabajo CSS del W3C publica desde 2015 (hasta ahora, cada dos años más o menos) las CSS Snapshot, que clasifican los borradores existentes en función de su grado de madurez e implementación. Actualmente (diciembre de 2021), la última versión publicada es CSS Snapshot 2021, publicada en diciembre de 2021. En este documento se listan las recomendaciones o borradores de recomendaciones CSS 3 que se consideran lo suficientemente maduras para sustituir los apartados correspondientes de la recomendación CSS 2.1.

Además del CSS Snapshot, la web del W3C ofrece información adicional y actualizada sobre CSS 3:

  • listado cronológico de las recomendaciones CSS vigentes y en elaboración
  • listado por grado de madurez de las recomendaciones CSS vigentes y en elaboración
  • listado completo de los borradores de editor existentes
  • índice de todos los términos definidos por las recomendaciones CSS: Propiedades y Descriptores

Al margen del W3C, algunas páginas recogen el estado de implementación de las diferentes recomendaciones en los diferentes navegadores:

Recomendaciones CSS 3 aprobadas

Actualmente (diciembre de 2021), el W3C ha aprobado las siguientes recomendaciones CSS 3:

  • junio de 2012: Media Queries, editada por Håkon Wium Lie y otros.
  • noviembre de 2013: CSS Style Attributes, editada por Tantek Çelik y Elika J. Etemad.
  • marzo de 2014: CSS Namespaces (2º ed), editada por Elika J. Etemad.
  • junio de 2018: CSS Basic User Interface Module Level 3 (CSS3 UI), editada por Tantek Çelik y Florian Rivoal.
  • septiembre de 2018: CSS Fonts Module Level 3, editada por John Daggett y otros.
  • noviembre de 2018: Selectors Level 3 (2ª ed), editada por Tantek Çelik y otros.
  • diciembre de 2019: CSS Writing Modes Level 3, editada por Elika J. Etemad y Koji Ishii.
  • diciembre de 2020: CSS Containment Module Level 1 (2ª ed), editada por Tab Atkins y Florian Rivoal.
  • febrero de 2021: CSS Cascading and Inheritance Level 3, editada por Elika J. Etemad y Tab Atkins. Aunque esta recomendación no ha sido retirada, ya no está incluida en la definición oficial de CSS de CSS Snapshot 2021, que incluye en su lugar la recomendación todavía en elaboración CSS Cascading and Inheritance Level 4.
  • agosto de 2021: CSS Color Module Level 3 (3ª ed), editada por Tantek Çelik y otros.
