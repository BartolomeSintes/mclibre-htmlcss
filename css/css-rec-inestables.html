La definición oficial de las hojas de estilo CSS se encuentra actualmente (febrero de 2023) en el documento CSS Snapshot 2021, publicado en diciembre de 2021. Esta lección enlaza las últimas versiones de las recomendaciones o de sus borradores que todavía no forman parte de forman esa definición. En la lección Recomendaciones CSS estables se enlazan las últimas versiones de los borradores que sí forman parte de la definición oficial de CSS.

En esta lección, los enlaces a los borradores se han dividido en tres grupos:

  • Recomendaciones CSS 3 en elaboración, pero relativamente estables. Se nombran en CSS Snapshot 2021 pero no forman parte de la definición oficial de CSS.
  • Recomendaciones CSS 3 en elaboración, que todavía necesitan refinarse y resolverse los problemas de interoperabilidad entre navegadores. Se nombran en CSS Snapshot 2021 pero no forman parte de la definición oficial de CSS.
  • Recomendaciones CSS 3 en elaboración, fuera de CSS Snapshot. No se nombran en CSS Snapshot 2021.
