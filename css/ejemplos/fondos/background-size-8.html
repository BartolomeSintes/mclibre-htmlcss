Como muestra un ejemplo posterior, si la página es una página vacía, los navegadores no interpretan el valor contain. Pero si la página tiene contenido, como en este ejemplo, o si el elemento tiene un tamaño definido, el valor contain hace que la imagen de fondo se ajuste de manera que su tamaño corresponda a la dimensión (horizontal o vertical) más pequeña.

Pero lo que el navegador tiene en cuenta es el contenido de la página, no el de la ventana. Por eso en este ejemplo, la imagen se hace más grande que en el ejemplo anterior, ya que este ejemplo contiene más texto.
