Visual Studio Code. Perfil recomendado para el curso Páginas web HTML y hojas de estilo CSS

VSC: Perfil

  • Instalación

Visual Studio Code permite utilizar perfiles, lo que permite cambiar de forma sencilla entre distintas configuraciones (extensiones instaladas, preferencias, tareas y atajos). Además, los perfiles se pueden compartir, lo que facilita la puesta a punto del editor en distintos ordenadores.

En esta lección se comenta cómo instalar un perfil que corresponde a la Configuración recomendada para seguir este curso. Consulte esa lección para conocer los detalles de la configuración.

Referencias:

  • Documentación oficial Profiles in Visual Studio Code

Instalación del perfil

Los perfiles de VSCode se pueden compartir mediante gists de GitHub. El perfil que corresponde a la configuración recomendada para seguir estos apuntes de HTML y CSS se encuentra disponible en https://gist.github.com/BartolomeSintes/7290b128a95f302d47a45e189236ce1a.

Una vez creado y seleccionado el perfil en Visual Studio Code, no se requieren ajustes adicionales.
