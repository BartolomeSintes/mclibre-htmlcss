Presentación

Me llamo Bartolomé Sintes Marco. Soy profesor de Informática en el Instituto de Educación Secundaria Abastos de Valencia (España).

En el curso 2023/24 dedicaré la primera mitad del curso (aproximadamente 45 horas de clase) a enseñar a crear páginas web (HTML+CSS) a alumnos del módulo Lenguajes de Marcas y Sistemas de Gestión de Información de 1º curso del ciclo formativo de grado superior Administración de Sistemas Informáticos en Red.

El módulo de Lenguaje de Marcas está orientado a alumnos sin conocimientos previos de creación de páginas web y se abordará el lenguaje de marcas HTML y las hojas de estilo CSS.

Estos apuntes no son una obra terminada, ni lo serán nunca debido a la naturaleza cambiante de la web. Cada curso intento actualizar y mejorar los contenidos, pero siempre me queda trabajo por hacer. Por otro lado, en clase no abordo todos los temas que se tratan en estos apuntes. La información adicional que contienen estos apuntes se ofrece al público en general y a los alumnos que quieran aprender más de lo abordado en clase.

Formato de estos apuntes

  • Las páginas de este curso son accesibles desde la página principal de los apuntes.
  • La página principal está dividida en seis apartados:
    • en el apartado Lecciones HTML se encuentran las lecciones sobre HTML, el lenguaje de marcas.
    • en el apartado Lecciones CSS se encuentran las lecciones sobre CSS, ordenadas por lecciones que corresponden más o menos a las lecciones sobre HTML. Estas lecciones están separadas en dos grupos, por un lado las propiedades CSS "estables", que funcionan prácticamente en todos los navegadores, y por otro lado las propiedades CSS "en desarrollo".
    • en el apartado Ejercicios se encuentran ejercicios agrupados por temas, que se corresponden más o menos con las lecciones HTML y CSS. Se puede ver el código fuente de todos los ejercicios propuestos y en muchos casos consultar comentarios de ayuda sobre los ejercicios propuestos.
    • en el apartado Exámenes se encuentran ejemplos de exámenes de años anteriores que se pueden utilizar como ejercicios adicionales (algunos ejercicios de exámenes se han incluido posteriormente en los ejercicios). No se incluyen soluciones de los exámenes propuestos.
    • en el apartado Software se enlazan lecciones dedicadas a la instalación y configuración de aplicaciones (Firefox, Visual Studio Code, etc.). Estas lecciones se encuentran realmente en otro apartado de mclibre.org denominado Informática general en el que a partir de este curso pretendo agrupar las lecciones dedicadas a programas específicos.

Programas empleados

En este curso se utilizarán programas libres multiplataforma.

Sistema operativo

Los ordenadores que utilizarán los alumnos tienen instalado Windows 10, por lo que he utilizado las versiones para ese sistema operativo de los programas empleados, pero el curso puede seguirse perfectamente desde cualquier distribución GNU/Linux o desde versiones anteriores de Windows.

Navegador

El navegador recomendado a los alumnos es Firefox, aunque el curso hace referencia también a Chrome. A principios de curso (septiembre de 2023), las últimas versiones disponibles eran Firefox 117 y Chrome 117. En los ordenadores de los alumnos estarán disponibles las últimas versiones (que se irán actualizando automáticamente a versiones posteriores).

En cdlibre.org hay una sección dedicada a navegadores web libres, con información detallada sobre las últimas versiones publicadas para Windows.

Editor

El editor recomendado en este curso es el editor Visual Studio Code, un editor libre, multiplataforma y que tiene un ritmo de desarrollo muy rápido, pero realmente este curso se puede seguir con cualquier editor de texto plano. A principios de curso (septiembre de 2023), la última versión disponible era Visual Studio Code 1.82. En los ordenadores de los alumnos estará disponible la última versión (que se irá actualizando automáticamente a versiones posteriores).

En cdlibre.org hay una sección dedicada a editores web libres, con información detallada sobre las últimas versiones publicadas para Windows.

Novedades

  • En el curso 2023/24 me gustaría abordar estos puntos:
    • Utilizaré perfiles de VSCode para facilitar la configuración del editor y su actualización a lo largo del curso.
    • En la parte de CSS, tomaré CSS Snapshot 2022 como referencia.
    • En CSS me gustaría abordar el posicionamiento flexbox y grid y dejar de dar el posicionamiento clásico.
    • Quiero dar mayor importancia al diseño de páginas para móviles
    • Me gustaría empezar a utilizar Git (concretamente, GitHub) como herramienta de colaboración y publicación.
    • Como todos los años, quiero seguir completando y actualizando la información ofrecida en todos los temas, incluyendo imágenes SVG explicativas.
  • En el curso 2022/23:
    • Tomé CSS Snapshot 2021 como referencia.
    • Pasé a utilizar el formato de colores hwb().
  • En el curso 2021/22:
    • Tomé CSS Snapshot 2018 como referencia. Me refiero a que en los apuntes hablo de CSS para hacer referencia a las recomendaciones y borradores CSS 3 que en CSS Snapshot 2018 se consideraban estables. Además, en las lecciones dedicadas a recomendaciones estables, si en clase vemos sólo una parte de las propiedades, empecé a dividir en dos partes las lecciones, en una página están las propiedades que vemos en clase y en otra página las que no.
    • Empecé a tratar en clase el posicionamiento flexbox
  • En el curso 2020/21:
    • No impartí este curso, así que apenas modifiqué los apuntes.
    • Eliminé las referencias al Microsoft Edge original
  • En el curso 2019/20:
    • Sustituí las referencias a la recomendación del W3C por la recomendación del WHATWG.
    • Mejoré la información sobre entidades numéricas y emojis
  • En el curso 2018/19:
    • Dejé de utilizar la sintaxis XHTML y adopté la sintaxis HTML, tanto en el código fuente de los apuntes como en los ejercicios propuestos
    • Reduje el número de etiquetas (de secciones y elementos en línea) vistas en clase, pero no pude tratar etiquetas HTML 5 no tratadas anteriormente (contenido incrustado, por ejemplo)
    • En CSS introduje algo de posicionamiento flexbox, pero seguí dando posicionamiento clásico.
  • En el curso 2017/18:
    • Dejé de utilizar Brackets y empecé a utilizar Visual Studio Code.
    • Actualicé las explicaciones a HTML 5.2 (nuevas etiquetas, etc.)
    • Eliminé las referencias a Internet Explorer 11 para hacer referencia a Microsoft Edge
    • Modifiqué el código fuente de los apuntes: los nombres de los ficheros y directorios contienen ahora guiones (-) en vez de guiones bajos (_), los ejemplos está ahora en ficheros aparte, no incluidos en los ficheros de explicaciones, unifiqué el sangrado de las páginas, eliminé las páginas xhtml que quedaban, etc.
    • LLegué a dar un par de clases sobre GitHub, pero al final del curso.
  • En el curso 2016/17:
    • Completé la información sobre las nuevas etiquetas HTML 5 que introduje el curso anterior
    • Sustituí muchas imágenes por imágenes SVG.
    • A final de curso, creé un repositorio GitHub para estos apuntes: https://github.com/BartolomeSintes/mclibre-htmlcss.
  • En el curso 2015/16:
    • Entré más a fondo en HTML 5 (concretamente, HTML 5 con sintaxis XHTML).
    • Con respecto a las hojas de estilo, seguí impartiendo CSS 2.1, aunque incluyendo algunos aspectos de CSS 3 que ya incorporan los navegadores actuales.
    • Incluí también lecciones dedicadas a gráficos SVG.
    • Para aligerar el material de consulta, eliminé las referencias al XHTML 1.0, al editor Amaya y a las versiones de Internet Explorer anteriores a IE 11, así como los ejercicios y exámenes anteriores anteriores al curso 2014/15 que estaban hechas en XHTML 1.0.
    • El juego de caracteres utilizado en todos los archivos pasó a ser utf-8, en vez del iso-8859-1 utilizado anteriormente.
    • También empecé a utilizar imágenes SVG en los propios apuntes (iconos, etc.)
  • En el curso 2014/15:
    • Empecé a impartir HTML 5 (más concretamente XHTML 5).
    • Para poder crear páginas HTML 5, tuve que empezar a utilizar un nuevo editor (Brackets) y por ello dediqué la mayor parte del tiempo a preparar explicaciones sobre Brackets y a preparar nuevos ejercicios en XHTML 5, pero sin entrar realmente a fondo en este nuevo lenguaje.
  • Hasta el curso 203/14:
    • En este módulo impartía XHTML 1.0 y CSS 2 (y algo de CSS 3) utilizando el editor Amaya, editor que dejó de desarrollarse en 2010. Al cabo de varios años había conseguido preparar un buen número de lecciones, ejercicios y exámenes, pero debido a la evolución permanente de la web, estos apuntes se estaban quedando obsoletos.
